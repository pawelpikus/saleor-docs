Creates/updates translations for attributes.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
attributeBulkTranslate(
  errorPolicy: ErrorPolicyEnum
  translations: [AttributeBulkTranslateInput!]!
): AttributeBulkTranslate
```

### Arguments

#### [`errorPolicy`](#) ([`ErrorPolicyEnum`](../../../developer/api-reference/enums/error-policy-enum))

Policies of error handling. DEFAULT: REJECT_EVERYTHING

#### [`translations`](#) ([`[AttributeBulkTranslateInput!]!`](../../../developer/api-reference/inputs/attribute-bulk-translate-input))

List of attributes translations.

### Type

#### [`AttributeBulkTranslate`](../../../developer/api-reference/objects/attribute-bulk-translate)

Creates/updates translations for attributes.

Requires one of the following permissions: MANAGE_TRANSLATIONS.
