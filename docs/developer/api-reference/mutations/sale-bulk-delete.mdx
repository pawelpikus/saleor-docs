Deletes sales.

Requires one of the following permissions: MANAGE_DISCOUNTS.

Triggers the following webhook events:

- SALE_DELETED (async): A sale was deleted.

```graphql
saleBulkDelete(
  ids: [ID!]!
): SaleBulkDelete
```

### Arguments

#### [`ids`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of sale IDs to delete.

### Type

#### [`SaleBulkDelete`](../../../developer/api-reference/objects/sale-bulk-delete)

Deletes sales.

Requires one of the following permissions: MANAGE_DISCOUNTS.

Triggers the following webhook events:

- SALE_DELETED (async): A sale was deleted.
