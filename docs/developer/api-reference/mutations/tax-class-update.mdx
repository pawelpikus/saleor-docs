Update a tax class.

Requires one of the following permissions: MANAGE_TAXES.

```graphql
taxClassUpdate(
  id: ID!
  input: TaxClassUpdateInput!
): TaxClassUpdate
```

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the tax class.

#### [`input`](#) ([`TaxClassUpdateInput!`](../../../developer/api-reference/inputs/tax-class-update-input))

Fields required to update a tax class.
