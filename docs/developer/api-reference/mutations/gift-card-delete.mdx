Delete gift card.

Requires one of the following permissions: MANAGE_GIFT_CARD.

Triggers the following webhook events:

- GIFT_CARD_DELETED (async): A gift card was deleted.

```graphql
giftCardDelete(
  id: ID!
): GiftCardDelete
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the gift card to delete.

### Type

#### [`GiftCardDelete`](../../../developer/api-reference/objects/gift-card-delete)

Delete gift card.

Requires one of the following permissions: MANAGE_GIFT_CARD.

Triggers the following webhook events:

- GIFT_CARD_DELETED (async): A gift card was deleted.
