Update stocks for product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantStocksUpdate(
  sku: String
  stocks: [StockInput!]!
  variantId: ID
): ProductVariantStocksUpdate
```

### Arguments

#### [`sku`](#) ([`String`](../../../developer/api-reference/scalars/string))

SKU of product variant for which stocks will be updated.

#### [`stocks`](#) ([`[StockInput!]!`](../../../developer/api-reference/inputs/stock-input))

Input list of stocks to create or update.

#### [`variantId`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant for which stocks will be updated.

### Type

#### [`ProductVariantStocksUpdate`](../../../developer/api-reference/objects/product-variant-stocks-update)

Update stocks for product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.
