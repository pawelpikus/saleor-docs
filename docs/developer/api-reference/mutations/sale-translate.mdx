Creates/updates translations for a sale.

DEPRECATED: this mutation will be removed in Saleor 4.0. Use `PromotionTranslate` mutation instead.

Requires one of the following permissions: MANAGE_TRANSLATIONS.

```graphql
saleTranslate(
  id: ID!
  input: NameTranslationInput!
  languageCode: LanguageCodeEnum!
): SaleTranslate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

Sale ID or SaleTranslatableContent ID.

#### [`input`](#) ([`NameTranslationInput!`](../../../developer/api-reference/inputs/name-translation-input))

Fields required to update sale translations.

#### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

Translation language code.

### Type

#### [`SaleTranslate`](../../../developer/api-reference/objects/sale-translate)

Creates/updates translations for a sale.
