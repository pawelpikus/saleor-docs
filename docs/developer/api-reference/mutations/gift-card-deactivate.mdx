Deactivate a gift card.

Requires one of the following permissions: MANAGE_GIFT_CARD.

Triggers the following webhook events:

- GIFT_CARD_STATUS_CHANGED (async): A gift card was deactivated.

```graphql
giftCardDeactivate(
  id: ID!
): GiftCardDeactivate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a gift card to deactivate.

### Type

#### [`GiftCardDeactivate`](../../../developer/api-reference/objects/gift-card-deactivate)

Deactivate a gift card.
