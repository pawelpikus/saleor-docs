Updates attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

Triggers the following webhook events:

- ATTRIBUTE_UPDATED (async): An attribute was updated.

```graphql
attributeUpdate(
  externalReference: String
  id: ID
  input: AttributeUpdateInput!
): AttributeUpdate
```

### Arguments

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of an attribute to update.

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of an attribute to update.

#### [`input`](#) ([`AttributeUpdateInput!`](../../../developer/api-reference/inputs/attribute-update-input))

Fields required to update an attribute.

### Type

#### [`AttributeUpdate`](../../../developer/api-reference/objects/attribute-update)

Updates attribute.
