Updates note of an order.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
orderNoteUpdate(
  note: ID!
  input: OrderNoteInput!
): OrderNoteUpdate
```

### Arguments

#### [`note`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the note.

#### [`input`](#) ([`OrderNoteInput!`](../../../developer/api-reference/inputs/order-note-input))

Fields required to create a note for the order.

### Type

#### [`OrderNoteUpdate`](../../../developer/api-reference/objects/order-note-update)

Updates note of an order.

Requires one of the following permissions: MANAGE_ORDERS.
