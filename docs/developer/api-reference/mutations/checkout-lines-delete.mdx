Deletes checkout lines.

Triggers the following webhook events:

- CHECKOUT_UPDATED (async): A checkout was updated.

```graphql
checkoutLinesDelete(
  id: ID
  linesIds: [ID!]!
  token: UUID
): CheckoutLinesDelete
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

The checkout's ID.

#### [`linesIds`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

A list of checkout lines.

#### [`token`](#) ([`UUID`](../../../developer/api-reference/scalars/uuid))

Checkout token.

DEPRECATED: this field will be removed in Saleor 4.0. Use `id` instead.

### Type

#### [`CheckoutLinesDelete`](../../../developer/api-reference/objects/checkout-lines-delete)

Deletes checkout lines.

Triggers the following webhook events:

- CHECKOUT_UPDATED (async): A checkout was updated.
