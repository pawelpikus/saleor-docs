Creates a value for an attribute.

Requires one of the following permissions: MANAGE_PRODUCTS.

Triggers the following webhook events:

- ATTRIBUTE_VALUE_CREATED (async): An attribute value was created.
- ATTRIBUTE_UPDATED (async): An attribute was updated.

```graphql
attributeValueCreate(
  attribute: ID!
  input: AttributeValueCreateInput!
): AttributeValueCreate
```

### Arguments

#### [`attribute`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

Attribute to which value will be assigned.

#### [`input`](#) ([`AttributeValueCreateInput!`](../../../developer/api-reference/inputs/attribute-value-create-input))

Fields required to create an AttributeValue.
