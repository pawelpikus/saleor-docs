Updates an existing variant for product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantUpdate(
  externalReference: String
  id: ID
  input: ProductVariantInput!
  sku: String
): ProductVariantUpdate
```

### Arguments

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of a product variant to update.

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product to update.

#### [`input`](#) ([`ProductVariantInput!`](../../../developer/api-reference/inputs/product-variant-input))

Fields required to update a product variant.

#### [`sku`](#) ([`String`](../../../developer/api-reference/scalars/string))

SKU of a product variant to update.

### Type

#### [`ProductVariantUpdate`](../../../developer/api-reference/objects/product-variant-update)

Updates an existing variant for product.

Requires one of the following permissions: MANAGE_PRODUCTS.
