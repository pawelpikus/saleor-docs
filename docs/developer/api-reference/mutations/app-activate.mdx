Activate the app.

Requires one of the following permissions: MANAGE_APPS.

Triggers the following webhook events:

- APP_STATUS_CHANGED (async): An app was activated.

```graphql
appActivate(
  id: ID!
): AppActivate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of app to activate.

### Type

#### [`AppActivate`](../../../developer/api-reference/objects/app-activate)

Activate the app.

Requires one of the following permissions: MANAGE_APPS.

Triggers the following webhook events:

- APP_STATUS_CHANGED (async): An app was activated.
