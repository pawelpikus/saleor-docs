Deletes menus.

Requires one of the following permissions: MANAGE_MENUS.

Triggers the following webhook events:

- MENU_DELETED (async): A menu was deleted.

```graphql
menuBulkDelete(
  ids: [ID!]!
): MenuBulkDelete
```

### Arguments

#### [`ids`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of menu IDs to delete.

### Type

#### [`MenuBulkDelete`](../../../developer/api-reference/objects/menu-bulk-delete)

Deletes menus.

Requires one of the following permissions: MANAGE_MENUS.

Triggers the following webhook events:

- MENU_DELETED (async): A menu was deleted.
