Deletes a product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productDelete(
  externalReference: String
  id: ID
): ProductDelete
```

### Arguments

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of a product to delete.

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product to delete.

### Type

#### [`ProductDelete`](../../../developer/api-reference/objects/product-delete)

Deletes a product.

Requires one of the following permissions: MANAGE_PRODUCTS.
