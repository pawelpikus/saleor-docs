Delete stocks from product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantStocksDelete(
  sku: String
  variantId: ID
  warehouseIds: [ID!]
): ProductVariantStocksDelete
```

### Arguments

#### [`sku`](#) ([`String`](../../../developer/api-reference/scalars/string))

SKU of product variant for which stocks will be deleted.

#### [`variantId`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of product variant for which stocks will be deleted.

#### [`warehouseIds`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

Input list of warehouse IDs.

### Type

#### [`ProductVariantStocksDelete`](../../../developer/api-reference/objects/product-variant-stocks-delete)

Delete stocks from product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.
