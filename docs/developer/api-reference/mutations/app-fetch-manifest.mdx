Fetch and validate manifest.

Requires one of the following permissions: MANAGE_APPS.

```graphql
appFetchManifest(
  manifestUrl: String!
): AppFetchManifest
```

### Arguments

#### [`manifestUrl`](#) ([`String!`](../../../developer/api-reference/scalars/string))

URL to app's manifest in JSON format.

### Type

#### [`AppFetchManifest`](../../../developer/api-reference/objects/app-fetch-manifest)

Fetch and validate manifest.

Requires one of the following permissions: MANAGE_APPS.
