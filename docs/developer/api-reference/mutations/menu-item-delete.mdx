Deletes a menu item.

Requires one of the following permissions: MANAGE_MENUS.

Triggers the following webhook events:

- MENU_ITEM_DELETED (async): A menu item was deleted.

```graphql
menuItemDelete(
  id: ID!
): MenuItemDelete
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a menu item to delete.

### Type

#### [`MenuItemDelete`](../../../developer/api-reference/objects/menu-item-delete)

Deletes a menu item.

Requires one of the following permissions: MANAGE_MENUS.
