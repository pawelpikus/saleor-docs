Deactivate gift cards.

Requires one of the following permissions: MANAGE_GIFT_CARD.

Triggers the following webhook events:

- GIFT_CARD_STATUS_CHANGED (async): A gift card was deactivated.

```graphql
giftCardBulkDeactivate(
  ids: [ID!]!
): GiftCardBulkDeactivate
```

### Arguments

#### [`ids`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of gift card IDs to deactivate.

### Type

#### [`GiftCardBulkDeactivate`](../../../developer/api-reference/objects/gift-card-bulk-deactivate)

Deactivate gift cards.

Requires one of the following permissions: MANAGE_GIFT_CARD.

Triggers the following webhook events:

- GIFT_CARD_STATUS_CHANGED (async): A gift card was deactivated.
