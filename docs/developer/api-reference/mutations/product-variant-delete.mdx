Deletes a product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantDelete(
  externalReference: String
  id: ID
  sku: String
): ProductVariantDelete
```

### Arguments

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of a product variant to update.

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of a product variant to delete.

#### [`sku`](#) ([`String`](../../../developer/api-reference/scalars/string))

SKU of a product variant to delete.

### Type

#### [`ProductVariantDelete`](../../../developer/api-reference/objects/product-variant-delete)

Deletes a product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.
