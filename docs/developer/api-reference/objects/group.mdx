Represents permission group data.

```graphql
type Group implements Node {
  id: ID!
  name: String!
  users: [User!]
  permissions: [Permission!]
  userCanManage: Boolean!
  accessibleChannels: [Channel!]
  restrictedAccessToChannels: Boolean!
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the group.

#### [`name`](#) ([`String!`](../../../developer/api-reference/scalars/string))

The name of the group.

#### [`users`](#) ([`[User!]`](../../../developer/api-reference/objects/user))

List of group users

Requires one of the following permissions: MANAGE_STAFF.

#### [`permissions`](#) ([`[Permission!]`](../../../developer/api-reference/objects/permission))

List of group permissions

#### [`userCanManage`](#) ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

True, if the currently authenticated user has rights to manage a group.

#### [`accessibleChannels`](#) ([`[Channel!]`](../../../developer/api-reference/objects/channel))

List of channels the group has access to.

#### [`restrictedAccessToChannels`](#) ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

Determine if the group have restricted access to channels.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
