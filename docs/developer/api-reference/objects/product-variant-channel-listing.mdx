Represents product variant channel listing.

```graphql
type ProductVariantChannelListing implements Node {
  id: ID!
  channel: Channel!
  price: Money
  costPrice: Money
  priorPrice: Money
  margin: Int
  preorderThreshold: PreorderThreshold
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the variant channel listing.

#### [`channel`](#) ([`Channel!`](../../../developer/api-reference/objects/channel))

The channel to which the variant listing belongs.

#### [`price`](#) ([`Money`](../../../developer/api-reference/objects/money))

The price of the variant.

#### [`costPrice`](#) ([`Money`](../../../developer/api-reference/objects/money))

Cost price of the variant.

#### [`priorPrice`](#) ([`Money`](../../../developer/api-reference/objects/money))

Prior price of the variant used for discount calculations.

Added in Saleor 3.21.

#### [`margin`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Gross margin percentage value.

Requires one of the following permissions: MANAGE_PRODUCTS.

#### [`preorderThreshold`](#) ([`PreorderThreshold`](../../../developer/api-reference/objects/preorder-threshold))

Preorder variant data.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
