Tax rate for a country. When tax class is null, it represents the default tax rate for that country; otherwise it's a country tax rate specific to the given tax class.

```graphql
type TaxClassCountryRate {
  country: CountryDisplay!
  rate: Float!
  taxClass: TaxClass
}
```

#### [`country`](#) ([`CountryDisplay!`](../../../developer/api-reference/objects/country-display))

#### [`rate`](#) ([`Float!`](../../../developer/api-reference/scalars/float))

Tax rate value.

#### [`taxClass`](#) ([`TaxClass`](../../../developer/api-reference/objects/tax-class))

Related tax class.
