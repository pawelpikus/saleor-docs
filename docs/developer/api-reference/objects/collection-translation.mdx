Represents collection translations.

```graphql
type CollectionTranslation implements Node {
  id: ID!
  language: LanguageDisplay!
  seoTitle: String
  seoDescription: String
  slug: String
  name: String
  description: JSONString
  descriptionJson: JSONString
  translatableContent: CollectionTranslatableContent
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the collection translation.

#### [`language`](#) ([`LanguageDisplay!`](../../../developer/api-reference/objects/language-display))

Translation language.

#### [`seoTitle`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated SEO title.

#### [`seoDescription`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated SEO description.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated collection slug.

Added in Saleor 3.21.

#### [`name`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated collection name.

#### [`description`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Translated description of the collection.

Rich text format. For reference see https://editorjs.io/

#### [`descriptionJson`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Translated description of the collection.

Rich text format. For reference see https://editorjs.io/

#### [`translatableContent`](#) ([`CollectionTranslatableContent`](../../../developer/api-reference/objects/collection-translatable-content))

Represents the collection fields to translate.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
