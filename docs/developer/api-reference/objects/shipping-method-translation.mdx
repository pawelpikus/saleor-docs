Represents shipping method translations.

```graphql
type ShippingMethodTranslation implements Node {
  id: ID!
  language: LanguageDisplay!
  name: String
  description: JSONString
  translatableContent: ShippingMethodTranslatableContent
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the shipping method translation.

#### [`language`](#) ([`LanguageDisplay!`](../../../developer/api-reference/objects/language-display))

Translation language.

#### [`name`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated shipping method name.

#### [`description`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Translated description of the shipping method.

Rich text format. For reference see https://editorjs.io/

#### [`translatableContent`](#) ([`ShippingMethodTranslatableContent`](../../../developer/api-reference/objects/shipping-method-translatable-content))

Represents the shipping method fields to translate.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
