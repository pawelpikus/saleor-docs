Creates product variants for a given product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
type ProductVariantBulkCreate {
  count: Int!
  productVariants: [ProductVariant!]!
  results: [ProductVariantBulkResult!]!
  bulkProductErrors: [BulkProductError!]!
  errors: [BulkProductError!]!
}
```

### Fields

#### [`count`](#) ([`Int!`](../../../developer/api-reference/scalars/int))

Returns how many objects were created.

#### [`productVariants`](#) ([`[ProductVariant!]!`](../../../developer/api-reference/objects/product-variant))

List of the created variants.This field will be removed in Saleor 4.0.

#### [`results`](#) ([`[ProductVariantBulkResult!]!`](../../../developer/api-reference/objects/product-variant-bulk-result))

List of the created variants.
