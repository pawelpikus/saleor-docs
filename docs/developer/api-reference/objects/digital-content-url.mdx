Represents a URL for digital content.

```graphql
type DigitalContentUrl implements Node {
  id: ID!
  content: DigitalContent!
  created: DateTime!
  downloadNum: Int!
  url: String
  token: UUID!
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the digital content URL.

#### [`content`](#) ([`DigitalContent!`](../../../developer/api-reference/objects/digital-content))

Digital content associated with the URL.

#### [`created`](#) ([`DateTime!`](../../../developer/api-reference/scalars/date-time))

Date and time when the digital content URL was created.

#### [`downloadNum`](#) ([`Int!`](../../../developer/api-reference/scalars/int))

Number of times digital content has been downloaded.

#### [`url`](#) ([`String`](../../../developer/api-reference/scalars/string))

URL for digital content.

#### [`token`](#) ([`UUID!`](../../../developer/api-reference/scalars/uuid))

UUID of digital content.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)
