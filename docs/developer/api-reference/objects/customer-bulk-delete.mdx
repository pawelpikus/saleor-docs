Deletes customers.

Requires one of the following permissions: MANAGE_USERS.

Triggers the following webhook events:

- CUSTOMER_DELETED (async): A customer account was deleted.

```graphql
type CustomerBulkDelete {
  count: Int!
  accountErrors: [AccountError!]!
  errors: [AccountError!]!
}
```
