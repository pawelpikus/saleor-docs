Represents page translations.

```graphql
type PageTranslation implements Node {
  id: ID!
  language: LanguageDisplay!
  seoTitle: String
  seoDescription: String
  slug: String
  title: String
  content: JSONString
  contentJson: JSONString
  translatableContent: PageTranslatableContent
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the page translation.

#### [`language`](#) ([`LanguageDisplay!`](../../../developer/api-reference/objects/language-display))

Translation language.

#### [`seoTitle`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated SEO title.

#### [`seoDescription`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated SEO description.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated page slug.

Added in Saleor 3.21.

#### [`title`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated page title.

#### [`content`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Translated content of the page.

Rich text format. For reference see https://editorjs.io/

#### [`contentJson`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Translated description of the page.

Rich text format. For reference see https://editorjs.io/

#### [`translatableContent`](#) ([`PageTranslatableContent`](../../../developer/api-reference/objects/page-translatable-content))

Represents the page fields to translate.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
