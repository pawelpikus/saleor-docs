Event sent when thumbnail is created.

```graphql
type ThumbnailCreated implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  id: ID
  url: String
  objectId: ID
  mediaUrl: String
}
```
