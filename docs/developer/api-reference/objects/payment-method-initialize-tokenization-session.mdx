Event sent when user requests a tokenization of payment method.

```graphql
type PaymentMethodInitializeTokenizationSession implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  user: User!
  channel: Channel!
  data: JSON
  paymentFlowToSupport: TokenizedPaymentFlowEnum!
}
```

### Fields

#### [`issuedAt`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Time of the event.

#### [`version`](#) ([`String`](../../../developer/api-reference/scalars/string))

Saleor version that triggered the event.

#### [`issuingPrincipal`](#) ([`IssuingPrincipal`](../../../developer/api-reference/unions/issuing-principal))

The user or application that triggered the event.

#### [`recipient`](#) ([`App`](../../../developer/api-reference/objects/app))

The application receiving the webhook.

#### [`user`](#) ([`User!`](../../../developer/api-reference/objects/user))

The user related to the requested action.

#### [`channel`](#) ([`Channel!`](../../../developer/api-reference/objects/channel))

Channel related to the requested action.

#### [`data`](#) ([`JSON`](../../../developer/api-reference/scalars/json))

Payment gateway data in JSON format, received from storefront.

#### [`paymentFlowToSupport`](#) ([`TokenizedPaymentFlowEnum!`](../../../developer/api-reference/enums/tokenized-payment-flow-enum))

The payment flow that the tokenized payment method should support.
