DEPRECATED: this type will be removed in Saleor 4.0. Use `PromotionTranslation` instead.

```graphql
type SaleTranslation implements Node {
  id: ID!
  language: LanguageDisplay!
  name: String
  translatableContent: SaleTranslatableContent
}
```
