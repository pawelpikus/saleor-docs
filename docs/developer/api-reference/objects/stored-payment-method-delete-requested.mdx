Event sent when user requests to delete a payment method.

```graphql
type StoredPaymentMethodDeleteRequested implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  user: User!
  paymentMethodId: String!
  channel: Channel!
}
```

### Fields

#### [`issuedAt`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Time of the event.

#### [`version`](#) ([`String`](../../../developer/api-reference/scalars/string))

Saleor version that triggered the event.

#### [`issuingPrincipal`](#) ([`IssuingPrincipal`](../../../developer/api-reference/unions/issuing-principal))

The user or application that triggered the event.

#### [`recipient`](#) ([`App`](../../../developer/api-reference/objects/app))

The application receiving the webhook.

#### [`user`](#) ([`User!`](../../../developer/api-reference/objects/user))

The user for which the app should proceed with payment method delete request.

#### [`paymentMethodId`](#) ([`String!`](../../../developer/api-reference/scalars/string))

The ID of the payment method that should be deleted by the payment gateway.

#### [`channel`](#) ([`Channel!`](../../../developer/api-reference/objects/channel))

Channel related to the requested delete action.
