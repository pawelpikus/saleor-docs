Represents category original translatable fields and related translations.

```graphql
type CategoryTranslatableContent implements Node {
  id: ID!
  categoryId: ID!
  seoTitle: String
  seoDescription: String
  slug: String
  name: String!
  description: JSONString
  descriptionJson: JSONString
  translation(languageCode: LanguageCodeEnum!): CategoryTranslation
  category: Category
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the category translatable content.

#### [`categoryId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the category to translate.

#### [`seoTitle`](#) ([`String`](../../../developer/api-reference/scalars/string))

SEO title to translate.

#### [`seoDescription`](#) ([`String`](../../../developer/api-reference/scalars/string))

SEO description to translate.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug to translate.

Added in Saleor 3.21.

#### [`name`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Name of the category translatable content.

#### [`description`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Category description to translate.

Rich text format. For reference see https://editorjs.io/

#### [`descriptionJson`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Description of the category.

Rich text format. For reference see https://editorjs.io/

#### [`translation`](#) ([`CategoryTranslation`](../../../developer/api-reference/objects/category-translation))

Returns translated category fields for the given language code.

- ##### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

A language code to return the translation for category.

#### [`category`](#) ([`Category`](../../../developer/api-reference/objects/category))

Represents a single category of products.
