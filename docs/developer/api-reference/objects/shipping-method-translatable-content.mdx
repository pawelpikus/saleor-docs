Represents shipping method's original translatable fields and related translations.

```graphql
type ShippingMethodTranslatableContent implements Node {
  id: ID!
  shippingMethodId: ID!
  name: String!
  description: JSONString
  translation(languageCode: LanguageCodeEnum!): ShippingMethodTranslation
  shippingMethod: ShippingMethodType
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the shipping method translatable content.

#### [`shippingMethodId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the shipping method to translate.

#### [`name`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Shipping method name to translate.

#### [`description`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Shipping method description to translate.

Rich text format. For reference see https://editorjs.io/

#### [`translation`](#) ([`ShippingMethodTranslation`](../../../developer/api-reference/objects/shipping-method-translation))

Returns translated shipping method fields for the given language code.

- ##### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

A language code to return the translation for shipping method.

#### [`shippingMethod`](#) ([`ShippingMethodType`](../../../developer/api-reference/objects/shipping-method-type))

Shipping method are the methods you'll use to get customer's orders to them. They are directly exposed to the customers.

Requires one of the following permissions: MANAGE_SHIPPING.
