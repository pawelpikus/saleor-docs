Contains all details related to the applied discount to the order.

```graphql
type OrderDiscount implements Node {
  id: ID!
  type: OrderDiscountType!
  name: String
  translatedName: String
  valueType: DiscountValueTypeEnum!
  value: PositiveDecimal!
  reason: String
  amount: Money!
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of discount applied.

#### [`type`](#) ([`OrderDiscountType!`](../../../developer/api-reference/enums/order-discount-type))

The type of applied discount: Sale, Voucher or Manual.

#### [`name`](#) ([`String`](../../../developer/api-reference/scalars/string))

The name of applied discount.

#### [`translatedName`](#) ([`String`](../../../developer/api-reference/scalars/string))

Translated name of the applied discount.

#### [`valueType`](#) ([`DiscountValueTypeEnum!`](../../../developer/api-reference/enums/discount-value-type-enum))

Type of the discount: fixed or percent

#### [`value`](#) ([`PositiveDecimal!`](../../../developer/api-reference/scalars/positive-decimal))

Value of the discount. Can store fixed value or percent value

#### [`reason`](#) ([`String`](../../../developer/api-reference/scalars/string))

Explanation for the applied discount.

Requires one of the following permissions: MANAGE_ORDERS.

#### [`amount`](#) ([`Money!`](../../../developer/api-reference/objects/money))

Returns amount of discount.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
