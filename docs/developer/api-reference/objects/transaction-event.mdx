Represents transaction's event.

```graphql
type TransactionEvent implements Node {
  id: ID!
  createdAt: DateTime!
  pspReference: String!
  message: String!
  externalUrl: String!
  amount: Money!
  type: TransactionEventTypeEnum
  createdBy: UserOrApp
  idempotencyKey: String
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the object.

#### [`createdAt`](#) ([`DateTime!`](../../../developer/api-reference/scalars/date-time))

Date and time at which a transaction event was created.

#### [`pspReference`](#) ([`String!`](../../../developer/api-reference/scalars/string))

PSP reference of transaction.

#### [`message`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Message related to the transaction's event.

#### [`externalUrl`](#) ([`String!`](../../../developer/api-reference/scalars/string))

The url that will allow to redirect user to payment provider page with transaction details.

#### [`amount`](#) ([`Money!`](../../../developer/api-reference/objects/money))

The amount related to this event.

#### [`type`](#) ([`TransactionEventTypeEnum`](../../../developer/api-reference/enums/transaction-event-type-enum))

The type of action related to this event.

#### [`createdBy`](#) ([`UserOrApp`](../../../developer/api-reference/unions/user-or-app))

User or App that created the transaction event.

#### [`idempotencyKey`](#) ([`String`](../../../developer/api-reference/scalars/string))

Idempotency key assigned to the event.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)

An object with an ID
