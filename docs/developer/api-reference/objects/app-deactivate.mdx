Deactivate the app.

Requires one of the following permissions: MANAGE_APPS.

Triggers the following webhook events:

- APP_STATUS_CHANGED (async): An app was deactivated.

```graphql
type AppDeactivate {
  appErrors: [AppError!]!
  errors: [AppError!]!
  app: App
}
```
