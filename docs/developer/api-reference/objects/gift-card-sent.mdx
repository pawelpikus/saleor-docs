Event sent when gift card is e-mailed.

```graphql
type GiftCardSent implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  giftCard: GiftCard
  channel: String
  sentToEmail: String
}
```

### Fields

#### [`issuedAt`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Time of the event.

#### [`version`](#) ([`String`](../../../developer/api-reference/scalars/string))

Saleor version that triggered the event.

#### [`issuingPrincipal`](#) ([`IssuingPrincipal`](../../../developer/api-reference/unions/issuing-principal))

The user or application that triggered the event.

#### [`recipient`](#) ([`App`](../../../developer/api-reference/objects/app))

The application receiving the webhook.

#### [`giftCard`](#) ([`GiftCard`](../../../developer/api-reference/objects/gift-card))

The gift card the event relates to.

#### [`channel`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which this gift card email was sent.

#### [`sentToEmail`](#) ([`String`](../../../developer/api-reference/scalars/string))

E-mail address to which gift card was sent.
