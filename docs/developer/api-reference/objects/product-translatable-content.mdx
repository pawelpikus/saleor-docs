Represents product's original translatable fields and related translations.

```graphql
type ProductTranslatableContent implements Node {
  id: ID!
  productId: ID!
  seoTitle: String
  seoDescription: String
  slug: String
  name: String!
  description: JSONString
  descriptionJson: JSONString
  translation(languageCode: LanguageCodeEnum!): ProductTranslation
  product: Product
  attributeValues: [AttributeValueTranslatableContent!]!
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the product translatable content.

#### [`productId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the product to translate.

#### [`seoTitle`](#) ([`String`](../../../developer/api-reference/scalars/string))

SEO title to translate.

#### [`seoDescription`](#) ([`String`](../../../developer/api-reference/scalars/string))

SEO description to translate.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug to translate.

Added in Saleor 3.21.

#### [`name`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Product's name to translate.

#### [`description`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Product's description to translate.

Rich text format. For reference see https://editorjs.io/

#### [`descriptionJson`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Description of the product.

Rich text format. For reference see https://editorjs.io/

#### [`translation`](#) ([`ProductTranslation`](../../../developer/api-reference/objects/product-translation))

Returns translated product fields for the given language code.

- ##### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

A language code to return the translation for product.

#### [`product`](#) ([`Product`](../../../developer/api-reference/objects/product))

Represents an individual item for sale in the storefront.

#### [`attributeValues`](#) ([`[AttributeValueTranslatableContent!]!`](../../../developer/api-reference/objects/attribute-value-translatable-content))

List of product attribute values that can be translated.
