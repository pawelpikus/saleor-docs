Deletes an app.

Requires one of the following permissions: MANAGE_APPS.

Triggers the following webhook events:

- APP_DELETED (async): An app was deleted.

```graphql
type AppDelete {
  appErrors: [AppError!]!
  errors: [AppError!]!
  app: App
}
```
