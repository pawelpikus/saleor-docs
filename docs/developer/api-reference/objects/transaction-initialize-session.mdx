Event sent when user starts processing the payment.

```graphql
type TransactionInitializeSession implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  transaction: TransactionItem!
  sourceObject: OrderOrCheckout!
  data: JSON
  merchantReference: String!
  customerIpAddress: String
  action: TransactionProcessAction!
  idempotencyKey: String!
}
```

### Fields

#### [`issuedAt`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Time of the event.

#### [`version`](#) ([`String`](../../../developer/api-reference/scalars/string))

Saleor version that triggered the event.

#### [`issuingPrincipal`](#) ([`IssuingPrincipal`](../../../developer/api-reference/unions/issuing-principal))

The user or application that triggered the event.

#### [`recipient`](#) ([`App`](../../../developer/api-reference/objects/app))

The application receiving the webhook.

#### [`transaction`](#) ([`TransactionItem!`](../../../developer/api-reference/objects/transaction-item))

Look up a transaction.

#### [`sourceObject`](#) ([`OrderOrCheckout!`](../../../developer/api-reference/unions/order-or-checkout))

Checkout or order

#### [`data`](#) ([`JSON`](../../../developer/api-reference/scalars/json))

Payment gateway data in JSON format, received from storefront.

#### [`merchantReference`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Merchant reference assigned to this payment.

#### [`customerIpAddress`](#) ([`String`](../../../developer/api-reference/scalars/string))

The customer's IP address. If not provided as a parameter in the mutation, Saleor will try to determine the customer's IP address on its own.

#### [`action`](#) ([`TransactionProcessAction!`](../../../developer/api-reference/objects/transaction-process-action))

Action to proceed for the transaction

#### [`idempotencyKey`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Idempotency key assigned to the transaction initialize.
