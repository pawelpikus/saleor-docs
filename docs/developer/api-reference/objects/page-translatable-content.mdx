Represents page's original translatable fields and related translations.

```graphql
type PageTranslatableContent implements Node {
  id: ID!
  pageId: ID!
  seoTitle: String
  seoDescription: String
  slug: String
  title: String!
  content: JSONString
  contentJson: JSONString
  translation(languageCode: LanguageCodeEnum!): PageTranslation
  page: Page
  attributeValues: [AttributeValueTranslatableContent!]!
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the page translatable content.

#### [`pageId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the page to translate.

#### [`seoTitle`](#) ([`String`](../../../developer/api-reference/scalars/string))

SEO title to translate.

#### [`seoDescription`](#) ([`String`](../../../developer/api-reference/scalars/string))

SEO description to translate.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug to translate.

Added in Saleor 3.21.

#### [`title`](#) ([`String!`](../../../developer/api-reference/scalars/string))

Page title to translate.

#### [`content`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Content of the page to translate.

Rich text format. For reference see https://editorjs.io/

#### [`contentJson`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Content of the page.

Rich text format. For reference see https://editorjs.io/

#### [`translation`](#) ([`PageTranslation`](../../../developer/api-reference/objects/page-translation))

Returns translated page fields for the given language code.

- ##### [`languageCode`](#) ([`LanguageCodeEnum!`](../../../developer/api-reference/enums/language-code-enum))

A language code to return the translation for page.

#### [`page`](#) ([`Page`](../../../developer/api-reference/objects/page))

A static page that can be manually added by a shop operator through the dashboard.

#### [`attributeValues`](#) ([`[AttributeValueTranslatableContent!]!`](../../../developer/api-reference/objects/attribute-value-translatable-content))

List of page content attribute values that can be translated.

### Interfaces

#### [`Node`](../../../developer/api-reference/interfaces/node)
