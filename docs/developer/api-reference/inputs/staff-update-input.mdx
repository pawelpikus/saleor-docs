Fields required to update a staff user.

```graphql
input StaffUpdateInput {
  firstName: String
  lastName: String
  email: String
  isActive: Boolean
  note: String
  metadata: [MetadataInput!]
  privateMetadata: [MetadataInput!]
  addGroups: [ID!]
  removeGroups: [ID!]
}
```

### Fields

#### [`firstName`](#) ([`String`](../../../developer/api-reference/scalars/string))

Given name.

#### [`lastName`](#) ([`String`](../../../developer/api-reference/scalars/string))

Family name.

#### [`email`](#) ([`String`](../../../developer/api-reference/scalars/string))

The unique email address of the user.

#### [`isActive`](#) ([`Boolean`](../../../developer/api-reference/scalars/boolean))

User account is active.

#### [`note`](#) ([`String`](../../../developer/api-reference/scalars/string))

A note about the user.

#### [`metadata`](#) ([`[MetadataInput!]`](../../../developer/api-reference/inputs/metadata-input))

Fields required to update the user metadata.

#### [`privateMetadata`](#) ([`[MetadataInput!]`](../../../developer/api-reference/inputs/metadata-input))

Fields required to update the user private metadata.

#### [`addGroups`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of permission group IDs to which user should be assigned.

#### [`removeGroups`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of permission group IDs from which user should be unassigned.
