### Fields

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the selected attribute.

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of this attribute.

#### [`values`](#) ([`[String!]`](../../../developer/api-reference/scalars/string))

The value or slug of an attribute to resolve. If the passed value is non-existent, it will be created.This field will be removed in Saleor 4.0.

#### [`dropdown`](#) ([`AttributeValueSelectableTypeInput`](../../../developer/api-reference/inputs/attribute-value-selectable-type-input))

Attribute value ID.

#### [`swatch`](#) ([`AttributeValueSelectableTypeInput`](../../../developer/api-reference/inputs/attribute-value-selectable-type-input))

Attribute value ID.

#### [`multiselect`](#) ([`[AttributeValueSelectableTypeInput!]`](../../../developer/api-reference/inputs/attribute-value-selectable-type-input))

List of attribute value IDs.

#### [`numeric`](#) ([`String`](../../../developer/api-reference/scalars/string))

Numeric value of an attribute.

#### [`file`](#) ([`String`](../../../developer/api-reference/scalars/string))

URL of the file attribute. Every time, a new value is created.

#### [`contentType`](#) ([`String`](../../../developer/api-reference/scalars/string))

File content type.

#### [`references`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

List of entity IDs that will be used as references.

#### [`richText`](#) ([`JSONString`](../../../developer/api-reference/scalars/jsonstring))

Text content in JSON format.

#### [`plainText`](#) ([`String`](../../../developer/api-reference/scalars/string))

Plain text content.

#### [`boolean`](#) ([`Boolean`](../../../developer/api-reference/scalars/boolean))

The boolean value of an attribute to resolve. If the passed value is non-existent, it will be created.

#### [`date`](#) ([`Date`](../../../developer/api-reference/scalars/date))

Represents the date value of the attribute value.

#### [`dateTime`](#) ([`DateTime`](../../../developer/api-reference/scalars/date-time))

Represents the date/time value of the attribute value.
