Define the filtering options for foreign key fields.

```graphql
input GlobalIDFilterInput {
  eq: ID
  oneOf: [ID!]
}
```

### Fields

#### [`eq`](#) ([`ID`](../../../developer/api-reference/scalars/id))

The value equal to.

#### [`oneOf`](#) ([`[ID!]`](../../../developer/api-reference/scalars/id))

The value included in.
