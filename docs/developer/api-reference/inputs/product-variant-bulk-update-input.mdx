Input fields to update product variants.

```graphql
input ProductVariantBulkUpdateInput {
  attributes: [BulkAttributeValueInput!]
  sku: String
  name: String
  trackInventory: Boolean
  weight: WeightScalar
  preorder: PreorderSettingsInput
  quantityLimitPerCustomer: Int
  metadata: [MetadataInput!]
  privateMetadata: [MetadataInput!]
  externalReference: String
  stocks: ProductVariantStocksUpdateInput
  channelListings: ProductVariantChannelListingUpdateInput
  id: ID!
}
```

### Fields

#### [`attributes`](#) ([`[BulkAttributeValueInput!]`](../../../developer/api-reference/inputs/bulk-attribute-value-input))

List of attributes specific to this variant.

#### [`sku`](#) ([`String`](../../../developer/api-reference/scalars/string))

Stock keeping unit.

#### [`name`](#) ([`String`](../../../developer/api-reference/scalars/string))

Variant name.

#### [`trackInventory`](#) ([`Boolean`](../../../developer/api-reference/scalars/boolean))

Determines if the inventory of this variant should be tracked. If false, the quantity won't change when customers buy this item. If the field is not provided, `Shop.trackInventoryByDefault` will be used.

#### [`weight`](#) ([`WeightScalar`](../../../developer/api-reference/scalars/weight-scalar))

Weight of the Product Variant.

#### [`preorder`](#) ([`PreorderSettingsInput`](../../../developer/api-reference/inputs/preorder-settings-input))

Determines if variant is in preorder.

#### [`quantityLimitPerCustomer`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Determines maximum quantity of `ProductVariant`,that can be bought in a single checkout.

#### [`metadata`](#) ([`[MetadataInput!]`](../../../developer/api-reference/inputs/metadata-input))

Fields required to update the product variant metadata.

#### [`privateMetadata`](#) ([`[MetadataInput!]`](../../../developer/api-reference/inputs/metadata-input))

Fields required to update the product variant private metadata.

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of this product variant.

#### [`stocks`](#) ([`ProductVariantStocksUpdateInput`](../../../developer/api-reference/inputs/product-variant-stocks-update-input))

Stocks input.

#### [`channelListings`](#) ([`ProductVariantChannelListingUpdateInput`](../../../developer/api-reference/inputs/product-variant-channel-listing-update-input))

Channel listings input.

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the product variant to update.
