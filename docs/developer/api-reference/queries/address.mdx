Look up an address by ID.

Requires one of the following permissions: MANAGE_USERS, OWNER.

```graphql
address(
  id: ID!
): Address
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an address.

### Type

#### [`Address`](../../../developer/api-reference/objects/address)

Represents user address data.
