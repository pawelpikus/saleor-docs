Look up a collection by ID or slug. If slugLanguageCode is provided, category will be fetched by slug translation. Requires one of the following permissions to include the unpublished items: MANAGE_ORDERS, MANAGE_DISCOUNTS, MANAGE_PRODUCTS.

```graphql
collection(
  id: ID
  slug: String
  slugLanguageCode: LanguageCodeEnum
  channel: String
): Collection
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the collection.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of the collection

#### [`slugLanguageCode`](#) ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Language code of the collection slug, omit to use primary slug.

Added in Saleor 3.21.

#### [`channel`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of a channel for which the data should be returned.

### Type

#### [`Collection`](../../../developer/api-reference/objects/collection)

Represents a collection of products.
