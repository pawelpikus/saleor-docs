Look up a channel by ID or slug.

```graphql
channel(
  id: ID
  slug: String
): Channel
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the channel.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of the channel.

### Type

#### [`Channel`](../../../developer/api-reference/objects/channel)

Represents channel.
