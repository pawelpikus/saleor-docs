Look up an app extension by ID.

Requires one of the following permissions: AUTHENTICATED_STAFF_USER, AUTHENTICATED_APP.

```graphql
appExtension(
  id: ID!
): AppExtension
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of the app extension.

### Type

#### [`AppExtension`](../../../developer/api-reference/objects/app-extension)

Represents app data.
