Look up a category by ID or slug.

```graphql
category(
  id: ID
  slug: String
  slugLanguageCode: LanguageCodeEnum
): Category
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the category.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

Slug of the category

#### [`slugLanguageCode`](#) ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Language code of the category slug, omit to use primary slug.

Added in Saleor 3.21.

### Type

#### [`Category`](../../../developer/api-reference/objects/category)

Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.
