Look up an order by ID or external reference.

```graphql
order(
  id: ID
  externalReference: String
): Order
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of an order.

#### [`externalReference`](#) ([`String`](../../../developer/api-reference/scalars/string))

External ID of an order.

Requires one of the following permissions: MANAGE_ORDERS.

### Type

#### [`Order`](../../../developer/api-reference/objects/order)

Represents an order in the shop.
