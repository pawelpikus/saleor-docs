Look up a page by ID or slug.

```graphql
page(
  id: ID
  slug: String
  slugLanguageCode: LanguageCodeEnum
): Page
```

### Arguments

#### [`id`](#) ([`ID`](../../../developer/api-reference/scalars/id))

ID of the page.

#### [`slug`](#) ([`String`](../../../developer/api-reference/scalars/string))

The slug of the page.

#### [`slugLanguageCode`](#) ([`LanguageCodeEnum`](../../../developer/api-reference/enums/language-code-enum))

Language code of the page slug, omit to use primary slug.

Added in Saleor 3.21.

### Type

#### [`Page`](../../../developer/api-reference/objects/page)

A static page that can be manually added by a shop operator through the dashboard.
